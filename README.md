<h1 align="center">Call Sentiment AI</h1>
<p align="center">AI-based system for analyzing customer call sentiment</p>

<hr>

<h2>📌 Overview</h2>
<p>
Call Sentiment AI is a web application that analyzes customer call recordings by converting speech into text and detecting sentiment. It classifies calls as <b>positive, negative, or neutral</b> to help understand customer feedback.
</p>

<h2>🚀 Features</h2>
<ul>
<li>Upload audio files for analysis</li>
<li>Convert speech to text</li>
<li>Perform sentiment analysis</li>
<li>Display results clearly</li>
<li>Simple and responsive UI</li>
</ul>

<h2>⚙️ How It Works</h2>
<ol>
<li>User uploads an audio file</li>
<li>Audio is converted into text</li>
<li>Text is analyzed for sentiment</li>
<li>Result is displayed (Positive / Negative / Neutral)</li>
</ol>
